# Repoman
A punk rock project mangager inspired from [@aa2006](https://github.com/aa2006)'s [proman](https://github.com/aa2006/proman).
## Building
```
git clone https://github.com/DamieFC/repoman
make all
```
## Usage
### Syntax
```
./repoman {option} {argument}
```
